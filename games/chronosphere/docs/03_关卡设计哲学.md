# 关卡设计哲学

本文件详述教学曲线、关卡类型、关卡拆解方法与样例分析。

教学曲线与关卡分层
- 引入（L1-4）：单一机制与简单目标，降低认知门槛
- 练习（L5-11）：加人对比、障碍与多目标，建立心智模型
- 组合（L12-15）：机制并置，要求多层次规划
- 挑战（L16-20）：有限资源与开放式谜题，考验最优解

关卡设计方法
- 模块化构建：将地形/目标/物体/卡牌视作可组合模块
- 解法可视化：在编辑器中记录至少 2 条不同解法
- 难度刻度：通过回溯次数、箱子数量、地图开阔度控制

样例详解
- L3：走廊引导回溯的教学意图与玩家操作流程
- L5：锚定与流动箱子的直接对比，强化心智模型
- L14：三机制综合考察，示例解法与常见误区解析
