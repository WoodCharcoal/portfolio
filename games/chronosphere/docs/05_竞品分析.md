# 竞品分析

目标：快速定位《时空回溯》在解谜市场中的差异化与机会点。

选取竞品（代表性）
1. Sokoban（经典推箱子）
   - 优势：规则极简、解法确定
   - 劣势：不可逆导致新手门槛高，玩法单一
   - 与本项目差异：《时空回溯》引入时间维度，提升容错与策略

2. Braid（独立解谜，时间操控）
   - 优势：将时间操控作为核心叙事与玩法元素，机制与剧情结合紧密
   - 劣势：偏线性、制作成本高
   - 与本项目差异：《时空回溯》将时间操控模块化为卡牌机制，便于教学与关卡设计

3. The Witness（开放式视觉谜题）
   - 优势：极高的观察与规则发现快感
   - 劣势：玩家高认知投入
   - 与本项目差异：《时空回溯》更强调有限资源下的策略规划与回溯决策

4. Baba Is You（规则操控）
   - 优势：通过改变规则创造巨大表达空间
   - 劣势：新手学习曲线陡峭
   - 与本项目差异：两者都强调“规则变动带来新解法”，但本项目通过视觉提示与渐进教学降低认知门槛

机会点与建议
- 将回溯与信标保护做成“教具式”的可视化，让玩家更容易通过观察形成正确心理模型
- 社区编辑器与关卡推荐能成为延长生命周期的关键
