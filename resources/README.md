## 数据格式

每个板块文件夹内包含：

- `README.md` — 板块说明与资源速览表
- `library.json` — 结构化资源索引，包含链接、标签、作者和备注

## 如何使用

1. 浏览各板块的 README 快速了解资源概览
2. 需要详细信息时查看对应文件夹的 library.json
3. 发现新资源时，按照现有 JSON 格式添加到对应分类即可
